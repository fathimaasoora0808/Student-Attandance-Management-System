# Student-Attandance-Management-System
Student Attendance Management System is a web-based application for managing student records and attendance. It supports role-based login for admins, lecturers, and students, attendance tracking, student management, and reports. 

Key Features:
1. User Authentication
2. Dashboard
3. Student Management
4. Attendance Tracking
5. Course & Batch Management
6. Reporting
7. Role-Based Access Control

 Technologies Used
* Frontend: HTML, CSS, Bootstrap
* Backend: PHP
* Database:MySQL

This project was developed to:
* Reduce manual effort in attendance tracking
* Improve data accuracy and accessibility
* Provide a scalable and maintainable solution for educational institutions
